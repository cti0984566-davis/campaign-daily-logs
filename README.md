# campaign-daily-logs

Public repository for campaign-based cyber threat daily logs.

## Purpose
- Accumulate cyber threat information by campaign
- Publish the process of building an automated collection and accumulation pipeline
- Support future campaign research and structured analysis

## Structure
- `schema/` : JSON schema definitions
- `logs/YYYY/MM/` : daily campaign logs
- `campaigns/` : campaign-focused structured summaries
- `scripts/` : helper scripts
- `docs/` : design notes and documentation

## Security Note
This is a public repository. Do not upload confidential, sensitive, personal, or non-public information.
